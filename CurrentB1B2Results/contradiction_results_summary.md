**Column Definitions**

- **Mode**: `c_full` = full memory corruption; `c_tail` = tail-only corruption
- **Offset**: stale memory offset (frames; 1/2/3)
- **Stale Persist.**: mean number of frames stale corruption persists across all scenes
- **Cum. Stale FP Length**: cumulative stale false-positive polyline length (proxy metric, meters)
- **Alpha Sep. Summary**: overall alpha separability summary (higher = model better distinguishes stale from fresh memory)
- **α_mean_affected**: mean alpha weight on corrupted/affected memory slots
- **α_mean_clean_recent**: mean alpha weight on clean recent memory slots
- **α_mean_preserved_recent**: mean alpha weight on preserved recent memory slots
- **AP@T**: average precision at matching threshold T (meters); AP = mean of AP@0.5/1.0/1.5
- **mAP**: mean AP across ped_crossing, divider, boundary categories
